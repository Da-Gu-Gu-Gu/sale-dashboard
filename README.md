## Run Locally

Clone the project

```bash
  git clone git@github.com:Da-Gu-Gu-Gu/sale-dashboard.git
```

Go to the project directory

```bash
  cd saleDashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
